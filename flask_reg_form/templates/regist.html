📘 What is HTML, CSS & JavaScript?

By Janvi Bhatt | June 5, 2025

🔤 What is HTML?

HTML structures the content of web pages using elements like headings, paragraphs, images, and more.

<h1>Hello World</h1>

🎨 What is CSS?

CSS styles your webpage with colors, fonts, spacing, layout, and responsiveness.

h1 {
  color: blue;
}

⚙️ What is JavaScript?

JavaScript makes the site interactive — clicking, animations, and logic handling.

document.querySelector('h1').style.color = 'red';
💡 Tip: Always use semantic HTML tags like <section>, <article>, etc.

📘 Glossary

  • Tag: HTML elements like <p> or <div>
  • Selector: CSS rule target like h1 or .box
  • DOM: Document Object Model used by JavaScript
